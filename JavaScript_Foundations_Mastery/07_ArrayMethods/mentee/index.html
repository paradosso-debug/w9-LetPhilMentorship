Array Methods – Practice

In this mini project you will practice using push, pop, shift, unshift, forEach, map, filter, and reduce.

  1. Open DevTools → Console.
  2. Open script.js and follow each STEP comment.
  3. Watch how arrays change as you use each method.

✨ Focus:
• Some methods change the original array (push, pop, shift, unshift)
• Some methods create a new array (map, filter)
• One method combines data into one final result (reduce)
